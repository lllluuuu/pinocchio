Getting Started

In the repository Pinocchio, you can find in the lab/ directory a makefile to create html pages. Ensure to have asciidoc and source-highlight installed on your computer: 
sudo apt-get install asciidoc source-highlight
After that, you will be able to generate the html pages with the Makefile. You can find the generated pages in the html directory. We advice to start with inverse-kinematics.html and then to continue with optimal-control.html. More to come...
